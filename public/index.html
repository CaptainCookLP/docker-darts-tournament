Schnellstart

  1. Öffne Settings und prüfe Defaults (X01, Legs/Sets pro Runde).
  2. Lege ein Turnier via API an (z. B. per curl/Postman) oder nutze deine bestehende UI.
  3. Starte das Turnier: POST /api/tournaments/:id/start.
  4. Zeig den Fortschritt in Bracket oder das Publikum-Overlay Overlay.

Hinweis: Diese Startseite ist bewusst minimal gehalten. Du kannst sie jederzeit ersetzen.
